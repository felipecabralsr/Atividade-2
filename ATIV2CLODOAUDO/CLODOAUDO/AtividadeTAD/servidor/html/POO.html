Quiz sobre Programação Orientada a Objetos


a) Classes são instâncias de objetos, e objetos são modelos de classes.
b) Classes são estruturas de dados que armazenam informações, e objetos são comportamentos que manipulam essas informações.
c) Classes são modelos que definem características e comportamentos, e objetos são instâncias desses modelos.


a) Herança múltipla é quando uma classe pode herdar atributos e métodos de várias outras classes, permitindo uma maior reutilização de código. Alguns exemplos de linguagens que suportam herança múltipla são C++ e Python.
b) Herança múltipla é quando uma classe pode ser instanciada múltiplas vezes, permitindo a criação de objetos com diferentes comportamentos. Linguagens como Java e C# suportam herança múltipla.
c) Herança múltipla é quando uma classe pode ter várias subclasses, permitindo uma hierarquia mais complexa de herança. Esta técnica é comum em linguagens como PHP e Ruby.


a) Sobrecarga de método é quando uma classe tem vários métodos com o mesmo nome, mas com diferentes assinaturas (número e tipo de parâmetros). Isso permite que a mesma ação seja realizada de maneiras diferentes, dependendo dos parâmetros fornecidos.
b) Sobrecarga de método é quando uma classe pode ter vários métodos com o mesmo nome e a mesma assinatura. Isso permite que a mesma ação seja realizada várias vezes na mesma classe.
c) Sobrecarga de método é quando uma classe tem vários métodos com nomes diferentes, mas com a mesma assinatura. Isso permite que a mesma ação seja realizada de maneiras diferentes, dependendo do nome do método.
